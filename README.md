[Bonjour Software](https://bonjour.software) pypahe is a **Py**thon **Pa**ckage **He**lper command-line tool.

## Requirements

- Python >= 3.9.4

## Usage

- find latest package version

```shell
./pypahe.sh latest boto3
1.17.62
```

- show available commands

```shell
./pypahe.sh  --help
```
